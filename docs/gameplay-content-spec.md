# Gameplay Content Spec

## Purpose
This document defines the waste-item library for the MVP so the game logic, visuals, and educational messaging all use the same source of truth.

## MVP Bin Categories
- General waste
- Paper
- Plastic
- Organic waste

## Content Rules
- Each waste item should appear with a short display label in-game.
- Each waste item belongs to exactly one bin in the MVP.
- The first version should prefer obvious household examples over edge cases.
- Ambiguous real-world recycling rules may vary by city, so the MVP should stay consistent with its own game rules.

## Spawn Variety Guidance
- Use all 32 selected MVP items, with 8 items per bin.
- Keep the category distribution balanced across a round.
- Avoid showing the same exact item twice in quick succession.
- Allow common items like bottles, banana peels, newspapers, and pizza boxes to appear slightly more often than niche items.
- Keep visually similar items spaced apart so the screen stays readable.

## Educational Messaging Guidance
- Show the teaching message immediately after a correct drop.
- Show the teaching message only the first time that specific item is sorted correctly during a run.
- Keep the message visible briefly, then fade it out.
- Use short and direct wording so it does not interrupt gameplay.
- Do not show long explanations during the 60-second round.

## Follow-Up Decisions
- Decide whether the visual cue names should map to emoji, illustrations, or icon assets.
- Decide the exact spawn weighting so common items appear a bit more often without feeling repetitive.
